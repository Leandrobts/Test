▸ Como Usar no PS4

Setup do Servidor Local

1. Copiar todos os arquivos HTML para uma pasta e rodar:

python3 -m http.server 80

2. No PS4: Configurações → Rede → Configurar conexão → Avançado

DNS Primário = <IP_DO_SEU_PC>

3. Configurar o DNS para redirecionar manuals.playstation.net para seu PC

4. No PS4: Configurações → Guia do Usuário (abre o browser interno)

5. Ou navegar diretamente para http://<IP>/index.html


Interpretando os Resultados

▸ Browser fecha/reinicia = BUG PRESENTE → exploit possível

▸ "Out of Memory" = BUG PRESENTE (crash controlado)

▸ Tela preta / freeze = BUG PRESENTE (possível hang)

▸ Completa sem crash = patched ou trigger precisa refinamento


Diagnóstico Avançado

Para confirmar crash em SceNKWebProcess:

Configurações → Sistema → Crash Report → Habilitar

Crash dump gerado em /dev_hdd0/user/crash/ (requer acesso ao sistema)
